## Installation

```json
{
    "require": {
        "linfo/laravel": "dev-master"
    }
}
```

or run `composer require linfo\laravel`

```php
Linfo\Laravel\LinfoServiceProvider::class,
```

```php
new \Linfo\Laravel\Models\Linfo()
```
