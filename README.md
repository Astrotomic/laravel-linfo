# laravel-linfo
This is a Laravel 5 Wrapper for the linfo package from jrgp - https://github.com/jrgp/linfo

```json
{
    "require": {
        "gummibeer/laravel-linfo": "dev-master"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/Gummibeer/laravel-linfo"
        }
    ]
}
```
